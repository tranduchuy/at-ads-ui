Hướng Dẫn Không Cho Phép Chặn IP Tùy Chỉnh

Bạn có thể nhập từng IP riêng lẻ, vui lòng điền mỗi IP 1 dòng

Ví dụ IP riêng lẻ:

127.0.0.1
127.0.0.2
127.0.0.3

Hoặc nhập từng nhóm IP, thêm dấu (*) tượng trưng cho nhóm IP không muốn chặn

Ví dụ nhóm IP:

IP4: 127.0.0.*
IP4: 127.0.*.*
